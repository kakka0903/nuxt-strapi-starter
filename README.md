## Installation

Run these commands and follow the instructions:
```bash
export GIGET_REGISTRY=https://raw.githubusercontent.com/kakka0903/nuxt-strapi-template/main
npx nuxi@latest init -t nuxt-strapi-starter
unset GIGET_REGISTRY
```
